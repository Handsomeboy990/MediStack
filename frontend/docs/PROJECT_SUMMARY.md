# Project summary, Frontend

## Goal

Provide the MediTrust web interface, tailored to the different roles of a health
center, for the cash desk, billing and stock tracking.

## Context

Built for the MTN Benin x Ministry of Health hackathon (Y'ello Care, 22 to 24
June 2026).

## Stack

- Next.js (App Router, TypeScript)
- Tailwind CSS
- Centralized HTTP client on top of `fetch`

## Roles covered

Director, Cash manager, Cashier, Storekeeper, HR. Each role has a dedicated
space.

## Current state

Skeleton of placeholder screens, working role selection home page, HTTP client
and error translation in place. Forms and lists will be wired to the API.
